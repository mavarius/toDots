## toDots

[![CircleCI](https://circleci.com/gh/mavarius/toDots.svg?style=svg)](https://circleci.com/gh/mavarius/toDots)

Replace text in a string with periods.

# Install

```bash
$ npm install toDots
```

## Usage

```js
const toDots = require('toDots')
toDots('SOME STRING') // --> '.... ......'
```
