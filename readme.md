## todots

[![CircleCI](https://circleci.com/gh/mavarius/todots.svg?style=shield)](https://circleci.com/gh/mavarius/todots)

Replace text in a string with periods.

## Install

```bash
$ npm install todots
```

## Usage

```js
const todots = require('todots')
todots('SOME STRING') // --> '.... ......'
```
